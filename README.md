# Memory-Matching-Game-

**Explanation** 
- When you first open the game, you see a player name input and a “Start Game” button.
- The board contains cards arranged in a grid.Each card has two sides. The goal is to match the colors. 
- You can click on a card to flip it and reveal its color.
- When all pairs are matched you win.
